Prompt bahasa inggris
1.	Buatkan program HTML untuk mengurutkan kata-kata bahasa inggris 
2.	Ada enam kata bahasa inggris seperti berikut ini [I] [am] [going] [to] [the] [market]
3.	Lakukan pengacakan atau random terhadap enam kata tersebut
4.	Mintalah murid untuk mengurutkan kata-kata tersebut. Jika murid mengurutkan seperti ini : [I-am-going-to-the-market] maka berilah feedback : “Hebat. Jawaban anda benar”. Jika selain itu, maka berilah feedback : “Maaf. Jawaban anda salah. Ulangi lagi”.
5.	Untuk mengurutkan kata-kata tersebut, maka buatkan teks area
6.	Pemindahan kata ke teks area dilakukan dengan cara drag dan drop
7.	Kata yang sudah dipindah
8.	Warna untuk skin soal berwarna hijau muda
9.	Teks area berwarna putih
10.	Background berwarna hijau
11.	Buatkan tombol hasil dan tombol reset
12.	Terima kasih
